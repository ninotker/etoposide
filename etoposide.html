Le modle utilis pour approximer au mieux le jeu de donnes intgre des variables catgorielles, permettant des ajustements spcifiques pour chaque groupe :

$$ F(t, \text{cat}) = (A + \delta_{\text{cat}}) \cdot (1 - e^{-(k + \epsilon_{\text{cat}})t}) $$

o :

$t$ reprsente le temps

$\text{cat}$ indique la catgorie (par exemple, Toposar exprience 1, Toposar exprience 2, NC exprience 1, NC exprience 2)

$A$ et $k$ sont les paramtres de base du modle

$\delta_{\text{cat}}$ et $\epsilon_{\text{cat}}$ sont les ajustements des paramtres $A$ et $k$, respectivement, pour chaque catgorie.

In[]:
# Importation de la fonction d'optimisation minimize de scipy
from scipy.optimize import minimize

# Prparation des catgories pour les donnes: 0 pour Toposar 1, 1 pour Toposar 2, etc.
category = np.array([0]*9 + [1]*9 + [2]*9 + [3]*9)

# Fonction modle gnralise prenant en compte les ajustements par catgorie
def generalized_model(params, t, cat):
    # Paramtres de base A et k
    A, k = params[:2]
    # Ajustements delta_A et delta_k pour chaque catgorie
    delta_A = params[2 + cat*2]
    delta_k = params[3 + cat*2]
    # Calcul de la valeur de la fonction avec ajustements
    A_cat = A + delta_A
    k_cat = k + delta_k
    return A_cat * (1 - np.exp(-k_cat * t))

# Fonction objective  minimiser: somme des carrs des rsidus
def objective_function(params, t, f_liberee, cat):
    # Calcul des prdictions pour chaque point de donnes et calcul des rsidus
    predictions = np.array([generalized_model(params, t[i], cat[i]) for i in range(len(t))])
    # Somme des carrs des diffrences entre prdictions et observations
    return np.sum((predictions - f_liberee) ** 2)

# Deviner initialement les paramtres pour l'optimisation
initial_guess = [45, 1] + [0, 0]*4  # A, k, et ajustements initiaux pour chaque catgorie

# Excution de l'optimisation pour minimiser la fonction objective
result = minimize(objective_function, initial_guess, args=(df["Temps"], df["F.libre"], category), method='L-BFGS-B')

# Affichage des paramtres optimiss
result.x

    return 42.31178488 * (1 - np.exp(-0.98304299 * t))

# Calcul des valeurs prdites par le modle pour les temps observs dans le DataFrame df
f_predicted = model_function(df["Temps"])

# Calcul des rsidus (diffrence entre les valeurs observes et prdites)
# puis levation au carr de ces diffrences pour obtenir les carrs des rsidus
squared_residuals = (df["F.libre"] - f_predicted) ** 2

# Somme des carrs des rsidus (SSR) pour valuer l'ajustement global du modle
ssr = np.sum(squared_residuals)
